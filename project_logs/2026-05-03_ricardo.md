## Key decisions locked in

| Decision | Choice | Reason |
|----------|--------|--------|
| Feedback type | Implicit with WMF confidence weights | Preserves rating signal without discarding low-rated interactions |
| Confidence weight | `c_ui = 1 + α × r_ui` | α tuned via Bayesian Optimization |
| Split | Leave-one-out by timestamp | Standard protocol (He et al. 2017) |
| Negatives | 99 per user, fixed seed=42 | Deterministic, comparable across all models and densities |
| Feature engineering | None | Isolates effect of interaction density |
| ID remapping | Contiguous 0-indexed | Required for PyTorch embedding tables |
